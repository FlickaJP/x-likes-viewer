# XLikeViewer

自分のX（Twitter）のいいねを丸ごとローカルに保存して、快適に閲覧・整理できる Windows 向けデスクトップアプリです。

## こんな悩みのある人へ

- 過去のいいねを見返したいのに、X上では古いいいねまでさかのぼるのが大変
- 何千・何万件のいいねを、画像つきで一覧して楽しみたい
- アカウント開設当時の、今の趣味に合わない古いいいねを整理（解除）したい

X本体では古いいいねへたどり着くのが困難ですが、このアプリで一度ローカルに集めてしまえば、古い順に並べて一番最初のいいねまで一発で見られます。

## できること

- 自分の全いいねをローカルに保存（本文・投稿者・画像・動画・GIF を原寸で）
- 高速なギャラリー表示（タイル / カード、サイズ変更、何万件でもスクロールで軽快）
- 検索・投稿者フィルタ・並び替え（いいね順 / 投稿日）・年月フィルタ
- お気に入り（★）を付けて絞り込み
- 拡大表示から投稿者ページや元ツイートをブラウザで開く
- 古いいいねを複数選択して一括解除（手元のアーカイブを残すかどうかも選べる）
- 新しいいいねの差分取得

## 動作環境

- Windows 10 / 11
- exe 版：ダウンロードしてそのまま実行（Python のインストール不要）
- ソース版：Python 3.10 以降 ＋ `pip install -r requirements.txt`

## 準備：X API のキーが必要です

このアプリは「自分のいいねを自分の認証で取得する」仕組みのため、**使う人それぞれが自分のX APIキーを用意する**必要があります（他人のキーは使えません）。

1. https://console.x.com で開発者登録し、アプリを作成します。
2. 「Keys & Tokens」で OAuth 1.0a の4つを取得します。
   - API Key（コンシューマーキー）
   - API Key Secret（コンシューマーシークレット）
   - Access Token（アクセストークン）
   - Access Token Secret（アクセストークンシークレット）
3. いいねの**解除機能**を使う場合は、Settings の「User authentication settings」で権限を **Read and Write** にしてから、Access Token を**再生成**してください（権限変更後に再生成しないと、読み取り専用のままになります）。
4. X API は従量課金（pay-per-use）です。自分のいいね取得は「Owned reads」で1件あたり約 $0.001。例えば2万件なら約 $20 程度が目安です。事前に Developer Console でクレジットの購入が必要です。

## 使い方

1. exe（またはソース版なら `python xlike_app.py`）を起動します。
2. 右上の ⚙ から、4つのキーを入力して保存します（初回のみ）。
3. 「↻ いいね取得」を押すと取り込みが始まります（初回は全件、以降は新着の差分のみ）。
4. 並び替え・検索・お気に入りで自由に閲覧します。
5. 解除したいときは「☑ 選択」を押して選択モードにし、消したいいいねをクリックで選び、「選択を解除」→ ダイアログで処理を選びます。

## 注意事項

- **APIキー（`config.json`）は絶対に公開・共有しないでください。** 漏れると悪用され、API利用料の請求などの被害につながります。
- いいねの解除はX本体に反映され、基本的に元には戻せません。
- API の利用料は自己負担です。料金体系はXの都合で変わることがあります。
- 本ツールの利用は**自己責任**で、Xの利用規約を守ってお使いください。
- exe は PyInstaller 製のため、ウイルス対策ソフトに誤検知されることがあります（無害です）。
